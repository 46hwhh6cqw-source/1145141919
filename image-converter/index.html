画像かんたん変換

HEIC（iPhone写真）をJPGへ、撮影日時・カメラ情報・GPSなどのEXIFを保持したまま変換します。 JPG / PNG / WebP / AVIF の相互変換、リサイズ、画質圧縮、位置情報の削除にも対応。

100% 端末内処理 · 画像はどこにも送信されません
出力形式
画質 92%（JPG / WebP のみ）
長辺サイズ（縮小のみ）
ファイルから起動したとき
ダブルクリックやショートカットへのドロップで開いたときの動作
自動保存の方法（保存先はダウンロードフォルダ）

クリックでファイル選択 ／ Ctrl+V でスクリーンショット等の貼り付け
HEIC · HEIF · JPG · PNG · WebP · GIF · BMP · AVIF ／ 複数ファイルOK
    ・変換はすべて端末内で完結します（サーバー送信なし・オフラインでも動作）。
    ・「ZIPでまとめて保存」では、各ファイルの更新日時に撮影日時を設定します。
    ・HEICの回転情報は画像自体に反映してから書き出すので、向きがズレることはありません。
    ・EXIFはJPG / PNG / WebPのどの出力形式でも引き継がれます（GIF・BMP入力にはEXIFがないため対象外）。
    ・共用PCなどインストールしたくない環境には ポータブル版（単一HTMLファイル） — 個人フォルダに置いてダブルクリックするだけで起動します。インストール不要・機能は同じです。
    アプリとしてインストールして使う（自分専用のPC向け）
    1. Chrome / Edge のアドレスバー右端のインストールアイコン、または上の「アプリとしてインストール」を押すと、独立ウィンドウのアプリになります。
    2. HEICファイルを右クリック →「プログラムから開く」→「画像かんたん変換」を選ぶと、このアプリで開けます。「常に使う」にすれば以後ダブルクリックで起動します。
    3. デスクトップやタスクバーのアプリアイコン（ショートカット）に画像をドラッグ＆ドロップするだけで自動変換・自動保存されます（動作は左の「ファイルから起動したとき」で変更できます。設定は大小どちらの画面とも共有です）。
    4. 画面右上の「大 / 小」で、フル表示と最小表示を切り替えられます。小画面では ⚙ から設定を開きます。

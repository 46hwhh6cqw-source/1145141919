# MicDoctor — 内蔵マイクの診断ファースト修復ツール

Dell Inspiron 7490 などの Windows ノートで「内蔵マイクが使えない」ときに、
**まず原因を切り分けてから、いちばん軽い対処で直す**ためのツールです。

渡された `DellMicRepair.ps1`（Realtek/Waves を丸ごと入れ直して2回再起動する版）は
最終手段としては有効ですが、多くの「マイクが壊れた」は実際にはソフト設定が原因で、
その場合ドライバーを消して大容量ダウンロード＆2回再起動するのは大げさで、失敗時の
リスクもあります。MicDoctor は「本当にドライバー入れ直しが要るのか」を先に判定します。

## 使い方

1. ZIP を展開します。
2. まず **`診断だけ.cmd`** をダブルクリック。
   - システムは一切変更しません。読み取りだけです。
   - どこに問題があるか（プライバシー設定 / サービス / デバイス無効化 / ドライバー異常）を表示します。
3. 直せる設定問題が見つかったら **`安全に修復.cmd`** をダブルクリック。
   - 管理者確認では「はい」。
   - **元に戻せる安全な修正だけ**を適用します。

## 診断する項目

- Windows のマイク プライバシー設定（システム全体 / ユーザー / デスクトップアプリ / グループポリシー）
- 音声サービス（Windows Audio, Audio Endpoint Builder）の稼働状態
- 録音（キャプチャ）デバイスの有無・有効/無効・エラー状態
- 音声ドライバー/コントローラの健全性（問題コード）
- 導入済み Realtek 音声ドライバーのバージョン（参考情報）

## `安全に修復.cmd` が行うこと（すべて可逆）

- マイクのプライバシーアクセスを「許可」に戻す
- 停止していた音声サービスを開始（自動起動に設定）
- 無効化されていた録音デバイスを有効化
- Windows Audio を再起動して設定を反映

## MicDoctor が **やらない** こと

- ドライバーの削除・アンインストール
- 数百MBのドライバーのダウンロード
- PC の再起動
- 個人ファイルの変更・削除

## 直らなかったときの案内（ツール内でも表示されます）

診断で「ドライバー/ハードの問題」と出た場合は、軽い順に：

1. デバイスマネージャーで音声デバイスを「アンインストール」→「ハードウェア変更のスキャン」
   （標準ドライバーが入り直る。ダウンロード不要・再起動不要）
2. それでも直らなければ Dell 公式ドライバーの入れ直し（渡された修復ツール等）
3. マイクがどのアプリでも一切反応せず、BIOS でも認識しないならハード故障 →
   Inspiron 7490 は内蔵マイクの分解交換になるため、Dell サポート/修理を検討

## ログ

`%LOCALAPPDATA%\MicDoctor\micdoctor_<日時>.log`

## 注意

Windows 10/11・PowerShell 5.1 以降を想定しています。会社/学校が管理する PC では、
グループポリシーでマイクがブロックされている場合があり、その場合は管理者に相談してください。
